#  Laundry Services Responsive Webpage

## Project Overview

This project is a responsive web page for a Laundry Services application. It was developed as part of a my  assignment to improve and understand FLEXBOX and MEDIA QUERY. You can seen different screen sizes such as desktop, tablet, and mobile devices.

The main goal is to fix responsiveness issues in the Hero Section and Navbar using **Flexbox** and **Media Queries**.


## Features

### Responsive Navbar

* Built using **Flexbox**
* Adjusts text size for tablet screens
* Hides navigation links on mobile view
* Displays only **logo and username** on smaller screens

### Hero Section

* Two sections: **Text (Left)** and **Image (Right)**
* Fully responsive layout
* Changes to **column layout on mobile devices**
* Adjusts text and image sizes for tablet and mobile screens

## Technologies Used

* HTML
* CSS
* Flexbox
* Media Queries

##  Responsive Design

###  Desktop View

* Full navigation menu visible
* Side-by-side hero layout

###  Tablet View

* Reduced font sizes
* Optimized image scaling

###  Mobile View

* Navigation links hidden
* Hero section stacked vertically
* Center-aligned content
